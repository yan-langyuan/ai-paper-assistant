## 三、交互设计规范

### 3.1 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（品牌蓝） | #2563EB | 按钮、链接、选中态 |
| 主色悬停 | #1D4ED8 | 按钮hover |
| 成功/低风险 | #16A34A | 🟢 绿色高亮、成功提示 |
| 警告/中度相似 | #F59E0B | 🟡 黄色高亮、提醒 |
| 危险/高度相似 | #DC2626 | 🔴 红色高亮、删除按钮 |
| 文字主色 | #1F2937 | 正文标题 |
| 文字辅色 | #6B7280 | 辅助说明、时间 |
| 背景色 | #F9FAFB | 页面底色 |
| 卡片色 | #FFFFFF | 卡片、输入框 |
| AI内容背景 | #EFF6FF | 摘要卡片、改写方案 |

### 3.2 间距规范

| 层级 | 值 | 用途 |
|------|------|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 组件内部元素间距 |
| md | 16px | 卡片内边距、段落间距 |
| lg | 24px | 模块间距 |
| xl | 32px | 页面区块间距 |
| 2xl | 48px | 页面顶部/底部留白 |

### 3.3 加载状态规范

| 场景 | 组件 | 说明 |
|------|------|------|
| 页面初始加载 | 骨架屏 | 模拟内容布局，灰色占位块 + shimmer动画 |
| 按钮操作中 | 按钮 Loading | 按钮文字变为"处理中..." + 左侧旋转图标 |
| AI生成中（摘要/改写） | 进度文案 | "正在生成摘要..." + 步骤提示 |
| 列表加载更多 | 底部 Loading | 滚动到底部显示"加载中..." |
| 文件上传 | 进度条 | 蓝色进度条 + 百分比数字 |

### 3.4 提示组件规范

| 类型 | 位置 | 持续时间 | 使用场景 |
|------|------|:---:|------|
| Toast成功 | 顶部居中 | 2秒 | 操作完成（复制/保存/导出） |
| Toast错误 | 顶部居中 | 3秒+手动关闭 | 操作失败 |
| 侧边栏提示 | 页面右侧悬浮 | 持续显示 | AI使用占比20%+ |
| 弹窗提醒 | 居中模态 | 手动关闭 | AI占比30%/50% |
| Tooltip | 跟随触发元素 | hover时显示 | 方案C学术变更提醒 |
